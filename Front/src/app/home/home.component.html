Olá!

Este projeto foi desenvolvido como parte da finalização do meu aprendizado no bootcamp de Java + Angular. Aqui, criei uma aplicação para o controle de transações financeiras, permitindo a inserção, exclusão e alteração de transações. Além disso, você pode conferir o gasto total e por categoria e obter outras informações sobre as transações.

Essa aplicação é desenvolvida com Spring Boot e Angular, combinando o back-end com o front-end para oferecer uma experiência intuitiva e eficiente. Além disso, a aplicação também faz conexão com o banco de dados MySQL, criando uma experiencia mais real do processo.

Estou muito orgulhosa do trabalho que realizei e animada para compartilhar essa jornada com você. Sinta-se à vontade para explorar e experimentar todas as funcionalidades.
